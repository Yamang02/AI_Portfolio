# P03: Main App Admin 진입점 제거

## 목표

Main App 라우터에 `/admin/*` 경로가 존재하지 않는다.
Header UI에 Admin으로 이동하는 설정 버튼이 존재하지 않는다.

## 구현 상세

### 접근

`App.tsx`에서 `/admin/*` 라우트와 `AdminApp` lazy import를 제거한다.
`Header.tsx`에서 설정 버튼(기어 아이콘)과 관련 click handler를 제거한다. 버튼 제거 후 Desktop/Mobile 레이아웃 균형을 검토하여 필요 시 간단한 UI 조정을 수행한다.

### 변경 범위

| 구분 | 파일 | 변경 내용 |
|------|------|----------|
| 수정 | `src/main/app/App.tsx` | `/admin/*` Route 및 `AdminApp` lazy import 제거, `AdminLoadingFallback` 제거 |
| 수정 | `src/main/widgets/header/ui/Header.tsx` | 설정 버튼 및 `navigate('/admin/settings')` handler 제거 (L54-58, L119-131, L210-218, L296-305) |

## 체크리스트

- [ ] `App.tsx`에 `AdminApp` import 및 `/admin/*` Route가 존재하지 않는다.
- [ ] `Header.tsx`에 `/admin/settings`로의 navigate 코드가 존재하지 않는다.
- [ ] Main App에서 `/admin/login`으로 직접 URL 입력 시 Admin 화면이 렌더링되지 않는다 (404 또는 Main 404 페이지).
- [ ] Main App 헤더가 설정 버튼 없이 정상 렌더링된다 (Desktop, Mobile 모두).
- [ ] `npm run build` 성공.
