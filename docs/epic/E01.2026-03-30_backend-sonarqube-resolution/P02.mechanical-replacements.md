## 구현 상세

### 접근 방식

- S6204(121건)는 프로젝트 전역 일괄 치환 — `collect(Collectors.toList())` → `toList()`, 이후 미사용 `Collectors` import 제거
- 나머지 규칙은 파일별로 처리, 동일 파일 내 여러 규칙은 한 번에 수정 (SQ-02-02 예외조항)
- 병렬 에이전트 활용 가능: S6204 전역치환 / 나머지 규칙별 분리

### 변경 범위

- S6204 영향 파일 전체 (121곳)
- 개별 규칙 해당 파일 (각 1~8곳)
