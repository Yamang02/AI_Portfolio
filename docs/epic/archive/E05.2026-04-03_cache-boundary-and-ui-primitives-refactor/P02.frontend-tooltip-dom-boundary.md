# P02: Tooltip/Header DOM 경계 리팩터

## 목표

`Tooltip`와 `HeaderIconButton` 조합에서 `<button>` 중첩이 발생하지 않으며 hydration 경고가 사라진다.

## 구현 상세

### 접근

- 디자인시스템 `Tooltip` 래퍼를 비상호작용 요소(`span`)로 변경한다.
- 툴팁 본문은 기존처럼 `role="tooltip"` 포털을 유지한다.
- Header 소비 코드는 최소 변경으로 회귀 범위를 줄인다.

### 변경 범위

| 파일 | 변경 내용 |
|------|-----------|
| `frontend/src/design-system/components/Tooltip/Tooltip.tsx` | wrapper 태그를 `span`으로 변경 |
| `frontend/src/design-system/components/Tooltip/Tooltip.module.css` | wrapper 스타일 유지 확인 |
| `frontend/src/main/widgets/header/ui/Header.tsx` | Tooltip+버튼 조합 회귀 확인 (필요 시 최소 수정) |

## 체크리스트

- [x] 콘솔에서 `button cannot be a descendant of button` 경고가 사라짐 (래퍼 비버튼화)
- [x] Header 테마 토글/메뉴 버튼 hover 및 click 동작이 기존과 동일함 (마크업만 변경)
- [x] `npm run build` 성공

## 검증 (세션)

- 명령: `npm run build` (`frontend/`)
- 결과: exit code `0`, `vite build` 완료 (~49s)
