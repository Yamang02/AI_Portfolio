# P03: missing-tokens-and-docs

## 목표

- z-index 토큰(`--ds-z-*`)이 정의되고 컴포넌트 내 하드코딩된 z-index 값이 교체된다
- motion/duration 토큰(`--ds-duration-*`, `--ds-ease-*`)이 정의되고 인라인 `transition` 값이 교체된다
- Storybook Token 스토리와 `theme-migration-guide.md`가 새 토큰 체계를 반영한다

## 구현 상세

### 접근

1. 컴포넌트 전수 grep으로 하드코딩된 `z-index`, `transition: Xs ease` 수집
2. z-index 레이어 정의 및 `globals.css` 추가
3. duration/ease 토큰 정의 및 `globals.css` 추가
4. 해당 컴포넌트 CSS Module 교체
5. Storybook `Tokens.stories.tsx`, `AllColors.stories.tsx` 업데이트
6. `theme-migration-guide.md` 갱신

### 신규 토큰

**Z-index**
```
--ds-z-base: 0
--ds-z-sticky: 100
--ds-z-overlay: 200
--ds-z-modal: 300
--ds-z-popover: 400
--ds-z-tooltip: 500
```

**Motion**
```
--ds-duration-fast: 150ms
--ds-duration-normal: 250ms
--ds-duration-slow: 400ms
--ds-ease-default: ease
--ds-ease-in: ease-in
--ds-ease-out: ease-out
--ds-ease-in-out: ease-in-out
```

### 변경 파일

| 파일 | 작업 |
|------|------|
| `frontend/src/design-system/styles/globals.css` | z-index / motion 토큰 추가 |
| `frontend/src/design-system/tokens/index.ts` | z-index / motion 토큰 export 추가 |
| z-index 사용 컴포넌트 CSS Module | `--ds-z-*` 교체 |
| transition 사용 컴포넌트 CSS Module | `--ds-duration-*` / `--ds-ease-*` 교체 |
| `frontend/src/design-system/tokens/Tokens.stories.tsx` | 신규 토큰 그룹 반영 |
| `frontend/src/design-system/tokens/AllColors.stories.tsx` | 신규 색상 토큰 반영 |
| `frontend/src/design-system/docs/theme-migration-guide.md` | 전면 갱신 |

## 체크리스트

- [ ] `globals.css`에 `--ds-z-*` 토큰 6개가 정의된다
- [ ] `globals.css`에 `--ds-duration-*` / `--ds-ease-*` 토큰이 정의된다
- [ ] `frontend/src` 전체에서 하드코딩된 `z-index: [0-9]` 패턴이 컴포넌트 CSS에서 0건이다
- [ ] Header / Modal / Tooltip의 z-index가 `--ds-z-*` 토큰을 사용한다
- [ ] Storybook Token 스토리에서 새 토큰 그룹이 렌더된다
- [ ] `theme-migration-guide.md`가 `--ds-*` / `--ui-*` 체계를 기준으로 작성된다
