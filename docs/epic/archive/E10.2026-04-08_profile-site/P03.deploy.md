# P03: 배포

## 목표

- `profile/dist/` 산출물이 별도 도메인으로 서빙된다.
- `frontend/` 배포 파이프라인에 영향을 주지 않는다.

## 구현 상세

- `profile/` 독립 빌드·배포 스텝을 CI/CD 파이프라인에 추가
- CloudFront + S3 또는 별도 정적 호스팅으로 서빙
- 백엔드 API 호출 없으므로 CORS 설정 변경 불필요

### 수정 파일

- CI/CD 파이프라인 파일 — profile 빌드·배포 스텝 추가

## 체크리스트

- [ ] 지정 도메인에서 프로필 페이지가 접근 가능하다
- [ ] `frontend/` 배포에 영향을 주지 않는다
- [ ] staging 환경에서 먼저 검증이 완료된다
