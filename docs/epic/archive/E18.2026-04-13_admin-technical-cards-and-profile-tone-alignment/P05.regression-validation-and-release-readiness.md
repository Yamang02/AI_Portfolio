# P05: regression-validation-and-release-readiness

## 목표

- Admin 기능과 Main 화면의 전체 흐름을 검증한다.
- 릴리즈 전 데이터 일관성/UI 동작 이상이 없음을 확인한다.

## 검증 시나리오

### 검증 범위

1. Admin 기능 검증
- 프로젝트 생성/수정/삭제
- Technical Card CRUD/정렬/표시 확인
2. Main 기능 검증
- 프로젝트 시나리오 로딩/에러/빈 상태
- `projectOverviewArticle`/Technical Card/링크/기술스택 표시

3. 동작 이상 검사
- Frontend 테스트(`npm run test:coverage`) 및 lint/build 확인
- 필요시 Sonar 점수와 새로운 이슈 triage

### 릴리즈 검증

- 기존 API/UI 일관성을 문서로 정리하는지 확인
- `readme` 제거 후 fallback이 깨져 보이지 않는지 확인
