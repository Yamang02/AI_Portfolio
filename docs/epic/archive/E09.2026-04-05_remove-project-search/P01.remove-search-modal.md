# P01: remove-search-modal

## 목표

- `ProjectSearchModal` 컴포넌트와 이를 여는 검색 버튼이 `ProjectsListPage`에서 제거된다.
- `techStackCategorization.ts` 및 `shared/utils/index.ts`의 해당 re-export가 제거된다.
- 빌드가 정상 통과하고 프로젝트 목록 페이지가 검색 관련 UI 없이 렌더링된다.

## 구현 상세

### 접근 방식

불필요한 코드 제거. 사용처를 역방향으로 추적하여 안전하게 삭제한다.

### 변경 범위

| 작업 | 파일 |
|------|------|
| 파일 삭제 | `frontend/src/main/pages/ProjectsListPage/components/ProjectSearchModal.tsx` |
| 파일 삭제 | `frontend/src/shared/utils/techStackCategorization.ts` |
| 수정 | `frontend/src/shared/utils/index.ts` — `techStackCategorization` re-export 제거 |
| 수정 | `frontend/src/main/pages/ProjectsListPage/ProjectsListPage.tsx` — 검색 버튼, `isSearchModalOpen` 상태, `ProjectSearchModal` 렌더링 제거 |

## 체크리스트

- [x] `ProjectSearchModal.tsx` 파일이 존재하지 않는다
- [x] `techStackCategorization.ts` 파일이 존재하지 않는다
- [x] `shared/utils/index.ts`에 `techStackCategorization` export가 없다
- [x] `ProjectsListPage`에 검색 아이콘 버튼이 렌더링되지 않는다
- [x] `ProjectsListPage`에 `isSearchModalOpen` 상태가 없다
- [x] `npm run build`가 오류 없이 완료된다
