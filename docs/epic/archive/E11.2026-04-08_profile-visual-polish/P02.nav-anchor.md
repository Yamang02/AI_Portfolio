# P02: 네비게이션-섹션 앵커 일치화

## 목표

네비게이션의 각 링크를 클릭하면 대응하는 섹션으로 부드럽게 스크롤된다.
`href="#"`인 상태가 없다.

## 구현 상세

### 접근 방식

각 `<section>`에 `id`를 부여하고, `<nav>` 내 `<a href>`를 해당 `id`와 매핑한다.
`<html>` 태그에 `scroll-behavior: smooth`를 적용한다.

### 앵커 매핑

| 네비게이션 텍스트 | 섹션 `id` | 대상 섹션 |
|---|---|---|
| Identity (→ 한국어 전환은 P03) | `#identity` | 관찰/사유/행동 섹션 |
| Works | `#works` | What I did 섹션 |
| Connect | `#connect` | Footer 또는 CTA 섹션 |

### 변경 파일

- `profile/index.html`
  - `<html>` 태그: `class`에 `scroll-smooth` 추가 (이미 있으면 생략)
  - Identity 섹션 `<section>`: `id="identity"` 추가
  - Works 섹션 `<section>`: `id="works"` 추가
  - Interests 섹션 `<section>`: `id="interests"` 추가 (필요 시 nav 항목 추가)
  - CTA 또는 Footer 섹션: `id="connect"` 추가
  - `<nav>` 링크: `href="#identity"`, `href="#works"`, `href="#connect"` 로 업데이트

## 체크리스트

- [ ] Identity 링크 클릭 시 관찰/사유/행동 섹션으로 스크롤된다
- [ ] Works 링크 클릭 시 프로젝트 섹션으로 스크롤된다
- [ ] Connect 링크 클릭 시 CTA 또는 Footer 영역으로 스크롤된다
- [ ] 스크롤이 부드럽게 동작한다 (`scroll-smooth`)
- [ ] `href="#"` 가 남아있는 nav 링크가 없다
