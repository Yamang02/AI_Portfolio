# P03: 영문 텍스트 한국어 전환

## 목표

페이지 내 영문 UI 텍스트가 한국어로 표기된다.
고유명사, 영문 태그, 연도 표기는 원형 유지된다.

## 구현 상세

### 접근 방식

`profile/index.html`에서 영문 UI 텍스트를 직접 치환한다.

### 전환 대상 목록

| 영문 원문 | 한국어 전환 |
|---|---|
| `Identity` (nav) | `정체성` |
| `Works` (nav) | `작업` |
| `Connect` (nav) | `연락` |
| `AI Portfolio` (nav 버튼) | `AI 포트폴리오` |
| `PORTFOLIO MMXXIV` | `포트폴리오 MMXXIV` |
| `What I did` (섹션 제목) | `내가 한 것들` |
| `Curated Projects 2022 — 2024` | `큐레이션된 프로젝트 2022 — 2024` |
| `View Project` (링크) | `자세히 보기` |
| `Now I'm interested in` (섹션 제목) | `지금 관심 있는 것들` |
| `AI Research` | `AI 연구` |
| `Sustainable Systems` | `지속 가능한 시스템` |
| `Digital Gardening` | `디지털 가드닝` |
| `A Synthesized Vision` (레이블) | `통합된 비전` |
| `이 모든 것이 담긴 결과물` | 유지 |
| `AI 포트폴리오 사이트 바로가기` | 유지 |
| `Built with intentionality` (footer) | `의도를 담아 만들었습니다` |
| `LinkedIn`, `Instagram`, `GitHub` | 유지 (고유명사) |

### 유지 대상 (변경 안 함)

- `Lee Jung-jun` (이름)
- `EDUCATION / TECH`, `ART / EXHIBITION`, `DESIGN / CURATION` (영문 태그)
- `Observation`, `Thought`, `Action` (괄호 내 영문 — 선택적 유지 검토)

### 변경 파일

- `profile/index.html`: 위 목록 기준 텍스트 치환

## 체크리스트
