# P03: 헤더 로고·페이지 라이프사이클

## 목표

- **로고**: `pathname === '/profile'` → 상단 스크롤, 그 외 → `navigate('/profile')`. (`/` 전용 분기 제거.)
- **`pageConfig.ts`**: `PAGE_CONFIG['/']` 항목 제거 또는 `/profile` 정책과 통합. `pageKey: 'home'` 제거 시 스크롤 저장 키 충돌이 없는지 확인.
- **`usePageLifecycle.ts`** 주석 예시에 `home`이 남아 있으면 갱신.
