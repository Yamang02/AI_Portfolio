# P04: SEO·사이트맵·도구·문서

## 목표

- **메타·canonical**: `pageMetaDefaults.home` 정리 — 제거 또는 프로필과 역할 분담. `defaultTitle`/`SeoHead` 동작이 빈 타이틀 등 예외 없이 유지되는지 확인.
- **JSON-LD**: `HomePage`에만 있던 `createOrganizationSchema()`, `createWebSiteSchema()`를 **`ProfilePage`의 `SeoHead`**(또는 상위에서 한 번만)로 이전. `createPersonSchema()`와 배열로 함께 전달 가능.
- **Sitemap**: `vite.config.ts`의 `routes`에서 **`/`** 항목 처리 — `/profile` 우선순위·changefreq 정책 반영, 중복 URL 정리.
- **스크립트**: `take-screenshots-after.ts`에서 랜딩 전용 캡처 제거 또는 경로 변경.
- **문서**: `developmentGuide.md`, SPA fallback 가이드 등 **`/` = 랜딩** 서술 갱신.

## 체크리스트

- [ ] 프로덕션 빌드 후 생성물·`sitemap.xml`에 기대 URL만 포함되는지 확인(로컬에서 빌드 스텝 실행).
- [ ] 선택: 인프라에서 **`/` → `/profile` HTTP 리다이렉트** 여부를 백로그/별 에픽에 기록.
