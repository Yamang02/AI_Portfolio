# P02: production-cf-alias-and-cert-cleanup

## 목표

CloudFront E384L5ALEPZ14U(production admin)의 alias가 `admin.yamang02.com` 단독으로 남고,
`yamangsolution.com` 관련 alias가 사라지며, 인증서가 `*.yamang02.com`(f3d39247)으로 교체된다.

## 구현 상세

### 현재 상태

- CF E384L5ALEPZ14U aliases: `admin.yamang02.com`, `yamangsolution.com`, `www.yamangsolution.com`
- 사용 중인 ACM cert: `c71ef080` (`yamangsolution.com` + `admin.yamang02.com` + `*.yamangsolution.com`)

### 변경 범위

- CF E384L5ALEPZ14U
  - aliases: `yamangsolution.com`, `www.yamangsolution.com` 제거 → `admin.yamang02.com` 단독
  - ACM cert: `c71ef080` → `f3d39247` (`*.yamang02.com`, 이미 ISSUED)

### 적용 방법

AWS 콘솔 또는 CLI로 CF 배포 설정 업데이트.
alias 제거와 인증서 교체를 **동일 업데이트에서 동시에** 적용해야 CF 검증 통과.

## 체크리스트

- [x] CF E384L5ALEPZ14U aliases에서 `yamangsolution.com`, `www.yamangsolution.com` 제거 확인
- [x] CF E384L5ALEPZ14U 인증서가 `f3d39247`(`*.yamang02.com`)으로 교체 확인
- [x] CF 배포 상태 `Deployed` 확인
- [ ] `https://admin.yamang02.com` 접속 200 확인
- [x] `yamangsolution.com`, `www.yamangsolution.com`이 이 CF를 더 이상 alias로 갖지 않음 확인
