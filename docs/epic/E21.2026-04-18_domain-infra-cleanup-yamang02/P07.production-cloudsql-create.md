# P07: production-cloudsql-create

## 목표

`ai-portfolio-pg-production` Cloud SQL 인스턴스가 GCP에 생성되고,
앱 유저와 DB가 준비되어 데이터 마이그레이션 수신 준비가 완료된다.

## 구현 상세

### 현재 상태

- production Cloud Run은 Railway(`turntable.proxy.rlwy.net:11437`)를 DB로 사용 중
- Terraform에 `module "postgres"` 정의되어 있으나 인스턴스 미생성

### 생성 스펙 (Terraform 정의 기준)

| 항목 | 값 |
|---|---|
| 인스턴스명 | `ai-portfolio-pg-production` |
| DB 버전 | POSTGRES_15 (staging 동일) |
| 티어 | db-f1-micro |
| 리전 | asia-northeast3 |
| 가용성 | ZONAL |
| 디스크 | 10GB |
| 백업 | 활성화, 7일 보관 |
| 삭제 방지 | 활성화 |

### 적용 방법

gcloud CLI로 직접 생성 후 Terraform state import (P05 import 작업과 연계).
