# P04: Easter egg 피처 코드 삭제

## 목표

`main/features/easter-eggs/` 디렉토리와 외부 참조 코드가 모두 제거된다.  
삭제 후 `tsc --noEmit`과 빌드가 오류 없이 통과한다.

## 구현 상세

### 삭제 대상

**피처 디렉토리 (전체 삭제):**

```
frontend/src/main/features/easter-eggs/
```

약 30개 파일 포함 — 컴포넌트, 훅, 스토어, 이펙트, 설정, 유틸, 타입 전체.

**외부 참조 파일별 변경:**

| 파일 | 변경 내용 |
|------|-----------|
| `main/pages/ChatPage/ChatPage.tsx` | easter-egg import 6개 제거, `useEasterEggEscapeKey()` 호출 제거, `useEasterEggTrigger(...)` 블록 제거, `<EasterEggLayer />` 제거, `<EasterEggProvider>` 래퍼 제거, `loadEasterEggConfig()` 초기화 블록 제거 |

| `shared/lib/hooks/useFeatureAvailability.ts` | `canUseEasterEgg` 필드 제거 |
