# P04: Identity 섹션 개편

## 목표

관찰/사유/행동의 추상적 철학 텍스트가 제거되고,
짧은 소개 단락 + 4개 역할 카드 구조로 교체된다.
방문자가 스크롤 없이 이 사람의 정체성을 구체적으로 파악할 수 있다.

## 구현 상세

### 접근 방식

기존 3열 철학 그리드를 제거하고 두 블록으로 교체한다.
1. **소개 단락** (full-width): 2~3문장 bio
2. **역할 카드 4개** (2×2 또는 4열 그리드): 역할명 + 한 줄 증거

### 콘텐츠 초안

**소개 단락**
> 교사로 시작해 개발자가 됐고, 예술가이기도 합니다.
> 세 정체성이 교차하는 지점에서 남들과 다른 시선으로 문제를 발견하고, 직접 만들어 증명합니다.

**역할 카드**

| 역할 | 한 줄 증거 |
|---|---|
| 교육자 | AI 시대 논리적 사고를 가르치는 강사 |
| 개발자 | 아이디어를 직접 코드로 구현하는 풀스택 개발자 |
| 예술가 | 성균관대 미술동아리 전시를 기획하고 아카이빙한 큐레이터 |
| 사업가 | 문제를 발견해 서비스로 만드는 실행가 |

### 레이아웃

```
[섹션 제목: 저는 이런 사람입니다]
[소개 단락 — max-w-2xl]
[역할 카드 × 4 — grid-cols-2 md:grid-cols-4]
```

### 변경 파일

- `profile/index.html`
  - Identity `<section>` 내부 전체 교체
  - 기존 `grid grid-cols-1 md:grid-cols-3` 3열 구조 → 소개 단락 + 4열 카드 구조

## 체크리스트

- [ ] 관찰/사유/행동 철학 텍스트가 페이지에 없다
- [ ] 소개 단락 2~3문장이 섹션 상단에 표시된다
- [ ] 역할 카드 4개(교육자/개발자/예술가/사업가)가 그리드로 렌더링된다
- [ ] 각 카드에 역할명 + 한 줄 설명이 표시된다
- [ ] 모바일에서 카드가 2열로 폴백된다
