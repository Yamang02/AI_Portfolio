# P01: 히어로 사진 배치

## 목표

히어로 섹션이 좌측 텍스트 + 우측 프로필 사진의 2열 구조로 렌더링된다.
모바일에서는 단열로 폴백된다.

## 구현 상세

### 접근 방식

히어로 `<section>`의 Flex 방향을 `flex-col md:flex-row`로 변경하고,
텍스트 블록과 이미지 블록을 나란히 배치한다.

### 변경 파일

- `profile/index.html`
  - 히어로 `<section>`: `flex-col items-start` → `flex-col md:flex-row md:items-center`
  - 기존 텍스트 블록을 `<div class="flex-1 ...">` 로 감싸기
  - 우측에 `<div class="...">` + `<img src="/img/hero_photo.jpg">` 추가

### 이미지 처리 규칙

- 비율: `aspect-[3/4]` 또는 `aspect-square` (사진 구도에 따라 결정)
- 크기: `w-full md:w-[420px]` 고정
- 스타일: `object-cover`, `rounded-xl`, 좌측과 간격 `md:ml-16`
- 모바일: `order-first md:order-last`로 모바일에서 사진이 위로 올라가는 옵션 고려

## 체크리스트

- [ ] 데스크톱(md 이상)에서 히어로가 텍스트 좌 / 사진 우 2열로 렌더링된다
- [ ] 모바일(md 미만)에서 단열로 표시된다
- [ ] 사진이 `hero_photo.jpg`를 올바르게 참조하고 깨지지 않는다
- [ ] 사진이 `object-cover`로 비율을 유지하며 잘린다
- [ ] 기존 텍스트(이름, 역할, 헤드라인) 레이아웃이 깨지지 않는다
