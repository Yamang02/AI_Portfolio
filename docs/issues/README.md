# 이슈 (Issues)

발견된 이슈, 버그 분석, 개선 사항 등을 기능별로 분류하여 관리합니다.

## 📁 기능별 분류

```
issues/
├── admin/              # 관리자 관련 이슈
├── billing/            # 결제/빌링 관련 이슈
├── cache/              # 캐시 관련 이슈
├── data-processing/    # 데이터 처리 관련 이슈
├── frontend/           # 프론트엔드 관련 이슈
├── sonarqube/          # SonarQube 분석 스냅샷
└── (루트)              # 일반 이슈 및 메타 문서
```

## 📝 이슈 문서 작성 가이드

각 이슈 문서는 다음을 포함해야 합니다:
- 문제 설명
- 원인 분석
- 해결 방안
- 영향 범위
- 관련 파일/코드
