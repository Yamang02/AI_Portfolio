## 로컬 개발

```bash
# 데이터베이스 및 Redis 시작
docker-compose up -d postgres redis

# 백엔드 시작
docker-compose up -d backend
```

접속 정보: 프론트엔드 `localhost:3000` / 백엔드 API `localhost:8080`
