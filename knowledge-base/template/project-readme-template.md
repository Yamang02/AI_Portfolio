### 로컬 실행 방법

#### 1. 저장소 클론
```bash
git clone [repository-url]
cd [project-name]
```

#### 2. 환경 변수 설정
```bash
# .env 파일 생성 및 설정
cp .env.example .env
# 필요한 환경 변수 입력
```

#### 3. 의존성 설치
```bash
# 프론트엔드
cd frontend
npm install

# 백엔드
cd backend
mvn install
```

#### 4. 서비스 실행
```bash
# Docker Compose로 전체 스택 실행
docker-compose up -d

# 또는 개별 실행
# 프론트엔드
cd frontend && npm run dev

# 백엔드
cd backend && mvn spring-boot:run
```

- 백엔드 API: http://localhost:8080
